Hoje 09-04-2014 aprendemos um pouco sobre Bootstrap.

Se você já quebrou a cabeça na formatação do CSS do seu website, principalmente na escolha da tipografia correta para os parágrafos, cabeçalhos, links e tantos outros componentes, o Bootstrap poderá lhe ser muito útil. Basicamente, trata-se de um emaranhado de links de CSS com algumas funções de Javascript embarcadas que te permitem fazer um website completo e bastante atual.

Mas nem só de CSS vive o Bootstrap. Ele possui DIVERSOS plugins de Javascript prontos para o uso que, com um simples CTRL+C e CTRL+V você poderá criar menus com dropdown, alertas, sliders, etc. Claro, a personalização fica a seu critério (os padrões disponibilizados pela ferramenta não são antiquados!).

O objetivo principal e lógico do Bootstrap é consumir o menor tempo possível no desenvolvimento de um website, seja ele uma página simples estática ou um grande portal dinâmico.
